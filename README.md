# 🚀 Project Management Tool

A modern **Project Management Dashboard** built using **HTML, CSS, and JavaScript**.
This tool helps users manage projects, tasks, and team activities efficiently through an interactive and responsive interface.

---

## ✨ Features

* 🔐 Login & Logout Interface
* 📋 Kanban Task Board (To Do, In Progress, Done)
* 📊 Real-time Analytics Dashboard
* 📅 Project Calendar
* 👥 Team Members Section
* 🔔 Notification Dropdown
* 📂 Project Management Section
* 🔍 Task Search Functionality
* 💾 LocalStorage Data Persistence
* 📱 Responsive UI Design

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage API
